# 🏢 AI Holding Office — Telegram Mini App

ИИ-команда для управления Telegram-холдингом. 9 сотрудников с уровнями, XP и обучением.

## 🚀 Деплой на GitHub Pages (бесплатно)

### Шаг 1 — Создать репозиторий
1. Зайти на [github.com](https://github.com) и войти в аккаунт
2. Нажать **New repository**
3. Назвать, например: `holding`
4. Поставить **Public**
5. Нажать **Create repository**

### Шаг 2 — Загрузить файл
1. На странице репозитория нажать **Add file → Upload files**
2. Перетащить файл `index.html`
3. Нажать **Commit changes**

### Шаг 3 — Включить GitHub Pages
1. Зайти в **Settings** репозитория
2. Слева найти **Pages**
3. В разделе **Source** выбрать: `Deploy from a branch`
4. Branch: `main`, папка: `/ (root)`
5. Нажать **Save**

Через 1–2 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/holding/
```

## 🤖 Привязка к Telegram-боту

### Шаг 1 — Создать бота
1. Открыть [@BotFather](https://t.me/BotFather) в Telegram
2. Написать `/newbot`
3. Придумать имя и юзернейм (должен заканчиваться на `bot`)
4. Сохранить токен (понадобится для других интеграций)

### Шаг 2 — Создать Mini App
В @BotFather написать `/newapp`:
- Выбрать своего бота
- Название: `AI Holding`
- Описание: `ИИ-команда вашего холдинга`
- Загрузить иконку (любой квадратный PNG, 640×640)
- URL: вставить ссылку с GitHub Pages

**Или проще** — через Menu Button:
1. `/mybots` → выбрать бота
2. **Bot Settings → Menu Button**
3. Вставить URL и название кнопки

### Шаг 3 — Открыть
Прямая ссылка для пользователей:
```
https://t.me/ВАШ_БОТ/app
```

## ⭐ Система уровней

- **+1 XP** за каждый ответ в чате
- **+2 XP** за обучение
- **5 XP** = новый уровень
- **5 уровней**: Новичок → Стажёр → Специалист → Эксперт → Мастер
